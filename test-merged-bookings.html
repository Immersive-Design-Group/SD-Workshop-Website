Merged Booking Slots - Test Page

This page demonstrates the new merged booking functionality that has been implemented in the reservation system.

New Features Implemented:

  • Consecutive Slot Merging: Multiple consecutive booking slots for the same person and activity are now displayed as a single merged card
  • Unified Information Display: Instead of showing separate cards for each 30-minute slot, one card shows the complete time range (e.g., "09:00-11:00" instead of separate "09:00-09:30", "09:30-10:00", etc.)
  • Bulk Deletion: When you click on a merged booking, you can delete all the consecutive slots at once instead of having to delete them one by one
  • Visual Enhancement: Merged bookings have special styling to indicate they span multiple time slots
  • Smart Detection: The system automatically detects consecutive bookings with the same person and purpose to merge them

How It Works:

  1. When you book multiple consecutive 30-minute slots (e.g., 09:00-09:30, 09:30-10:00, 10:00-10:30, 10:30-11:00) for the same activity
  2. The system automatically merges them into one visual card showing "09:00-11:00"
  3. All the booking information (name, purpose, etc.) is displayed once instead of repeated
  4. When you click to delete, all the consecutive slots are deleted together
